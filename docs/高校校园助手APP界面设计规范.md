# 高校校园助手 APP（HarmonyOS NEXT）界面设计规范

本设计规范基于 HarmonyOS NEXT 开发规范，贴合移动端操作习惯，采用简约校园风设计，以清新蓝为主色调、浅灰为背景色、暖橙为提醒色，整体布局简洁直观，功能模块划分清晰，适配 2-3 人学生团队 2 个月开发周期，同时覆盖课程教学大纲中的 UI 布局、组件开发、页面路由等核心技术点。设计稿包含启动页、引导页、登录 / 注册页、首页、校园信息、学习助手、生活服务、我的八大核心页面，各页面遵循统一的设计规范，保证视觉一致性和操作连贯性。

## 一、设计基础规范

### 1.1 配色规范

| 颜色类型 | 色值 | 应用场景 |
|---------|------|---------|
| 主色调 | #1677FF（鸿蒙蓝） | 导航栏、Tab 栏、按钮、选中状态、标题文字 |
| 辅助色 1 | #FF7D00（暖橙） | 提醒标识、待办标记、进度条、重点按钮 |
| 辅助色 2 | #00B42A（清新绿） | 完成状态、成功提示、正常状态标识 |
| 背景色 | #F5F7FA（浅灰） | 页面整体背景、模块分隔背景 |
| 文字主色 | #1D2129（深黑） | 正文、标题、重要信息文字 |
| 文字辅色 | #86909C（浅灰） | 辅助说明、占位符、时间 / 次要信息文字 |
| 边框色 | #E5E6EB（淡灰） | 模块边框、输入框边框、分割线 |

### 1.2 字体规范

采用 HarmonyOS NEXT 系统默认字体 **HarmonyOS Sans**，保证跨设备兼容性，文字大小及行高遵循移动端视觉舒适度要求：

| 文字类型 | 字号（sp） | 行高 | 字重 | 应用场景 |
|---------|-----------|------|------|---------|
| 大标题 | 32 | 1.2 倍 | 700 | 页面主标题、APP 名称 |
| 中标题 | 28 | 1.3 倍 | 600 | 模块标题、卡片标题 |
| 正文 | 26 | 1.5 倍 | 400 | 功能说明、内容文字、按钮文字 |
| 辅助文字 | 24 | 1.4 倍 | 400 | 时间、备注、占位符、底部说明 |

### 1.3 组件规范

- **圆角**：所有卡片、按钮、图片、输入框统一使用 **8px 圆角**，贴合移动端主流设计风格；
- **间距**：页面内元素间距为 16px，模块间距为 24px，页面左右边距为 20px，保证界面整洁不拥挤；
- **按钮**：默认按钮高度 48px，圆角 8px，文字居中；小按钮高度 36px，适配紧凑布局；
- **输入框**：高度 48px，圆角 8px，内边距左右 16px，默认显示浅灰色占位符；
- **卡片**：宽度 100%（适配页面），阴影为轻微高斯模糊（opacity:0.05），突出层次感无厚重感。

### 1.4 适配规范

适配 HarmonyOS NEXT 主流移动端设备（手机），屏幕尺寸覆盖 360×780 ~ 412×915，采用弹性布局（Row/Column）+ 相对布局（RelativeContainer），保证所有页面元素自适应，无内容遮挡、拉伸变形问题，贴合课程大纲中 ArkUI 布局教学要求。

## 二、核心页面设计

### 2.1 启动页

**页面尺寸**：全屏适配，无状态栏、导航栏

**设计元素**：

- **背景**：渐变鸿蒙蓝（#1677FF → #4096FF）全屏背景，无多余元素；
- **中心区域**：
  - 上方：APP logo（简约校园建筑 + 书本图标，白色）；
  - 中间：APP 名称「高校校园助手」（白色，36sp，字重 700）；
  - 下方：小字「HarmonyOS NEXT 版」（白色，24sp，字重 400）；
- **底部**：距离底部 30px，显示开发团队名称（白色，22sp）+ 加载动画（白色环形进度条，默认 3 秒完成）。

**交互逻辑**：
- 启动页停留 3 秒，自动跳转到引导页（首次打开）/ 首页（非首次打开）；
- 支持点击屏幕任意位置跳过启动页，贴合移动端操作习惯。

**技术要点**：
- 沉浸式窗口设计（贴合 Stage 模型窗口沉浸式能力教学）；
- 动画组件使用（环形进度条、页面跳转过渡）。

### 2.2 引导页（首次打开 APP 显示）

**页面结构**：左右滑动式引导，共 3 页，底部显示页码指示器，最后一页含「立即进入」按钮

**单页布局（统一框架）**：
- **背景**：#F5F7FA 浅灰色背景；
- 上半部分（60%）：引导图（简约扁平风，贴合页面主题）；
- 下半部分（40%）：
  - 标题：居中，32sp，鸿蒙蓝，字重 700；
  - 说明：标题下方，26sp，文字主色，行高 1.5 倍，居中；
  - 页码：底部居中，3 个圆形指示器，当前页为鸿蒙蓝填充，其余为淡灰色描边；
  - 按钮：仅第三页显示「立即进入」按钮（鸿蒙蓝，48px 高，圆角 8px，文字白色 26sp），位于页码上方，宽度 80%。

**三页主题内容**：

| 页码 | 标题 | 说明 | 引导图 |
|------|------|------|--------|
| 第一页 | 校园信息，实时掌握 | 校园公告、资讯、地图，一手了解校园动态 | 校园建筑 + 公告图标 |
| 第二页 | 学习助手，高效备考 | 课程表、成绩查询、图书馆检索，助力学业提升 | 书本 + 课程表图标 |
| 第三页 | 生活服务，贴心相伴 | 失物招领、校园美食、音乐播放，丰富校园生活 | 美食 + 爱心图标 |

**交互逻辑**：
- 左右滑动切换页面，滑动时有淡入淡出过渡动画；
- 点击页码指示器可快速跳转到对应页面；
- 点击第三页「立即进入」按钮，跳转到登录 / 注册页；
- 支持侧滑返回启动页（仅第一页）。

**技术要点**：
- 滑动组件使用、页面过渡动画（贴合导航和页面路由教学）；
- 按钮组件事件绑定、指示器组件状态切换。

### 2.3 登录 / 注册页（合并布局，切换标签）

**页面结构**：顶部状态栏（透明）+ 中间核心区域 + 底部辅助区域，整体居中布局，背景为 #F5F7FA 浅灰

**核心区域（宽度 90%，居中）**：

- **顶部标签栏**：
  - 两个标签：「登录」「注册」，左右平分宽度，文字 28sp；
  - 选中标签：文字为鸿蒙蓝，底部显示 2px 鸿蒙蓝下划线；未选中：文字为文字辅色，无下划线；
  - 支持点击标签切换登录 / 注册表单，无刷新切换，表单淡入显示。

- **表单区域**（标签栏下方，间距 24px）：

  **登录表单**：
  - 学号输入框：左侧学号图标（鸿蒙蓝），占位符「请输入你的学号」；
  - 密码输入框：左侧密码图标（鸿蒙蓝），占位符「请输入你的密码」，右侧显示 / 隐藏密码按钮（眼睛图标）；
  - 记住密码：左侧复选框（HarmonyOS 原生），右侧文字「记住密码」（24sp，文字辅色），位于输入框下方左侧；
  - 忘记密码：文字「忘记密码」（24sp，鸿蒙蓝），位于输入框下方右侧，点击跳转到找回密码页；
  - 登录按钮：宽度 100%，高度 48px，鸿蒙蓝背景，白色文字「登录」（26sp），圆角 8px。

  **注册表单**（在登录表单基础上增加）：
  - 姓名输入框、学号输入框、密码输入框、确认密码输入框（均带原生图标，占位符对应）；
  - 验证码输入框：左侧验证码图标，右侧「获取验证码」按钮（小按钮，暖橙色，文字白色 24sp）；
  - 注册协议：左侧复选框，右侧文字「我已阅读并同意《用户注册协议》」（22sp，文字辅色），协议文字为鸿蒙蓝，点击可查看协议弹窗；
  - 注册按钮：同登录按钮样式，文字「注册」。

**底部辅助区域**：
- 位置：距离页面底部 30px，居中；
- 内容：「暂无账号？」（24sp，文字辅色）+「立即注册」（24sp，鸿蒙蓝）（登录页）/「已有账号？」+「立即登录」（注册页）；
- 快捷登录：辅助区域上方，间距 20px，显示「其他登录方式」（24sp，文字辅色）+ 微信 / QQ 图标（圆形，36px 大小），图标居中排列，间距 16px。

**交互逻辑**：
- 点击标签栏快速切换登录 / 注册表单，表单内容实时切换，无页面跳转；
- 输入框获取焦点时，边框变为鸿蒙蓝，弹出系统软键盘，页面自适应上移，无遮挡；
- 按钮未激活状态：背景色 #C6E2FF，文字 #86909C；激活状态（所有必填项填写完成）：背景色 #1677FF，文字白色；
- 点击登录 / 注册按钮，验证表单信息（简易前端验证），验证通过跳转到首页，同时保存用户信息到本地。

**技术要点**：
- 输入框、复选框、按钮等原生组件使用及事件绑定（贴合组件教学）；
- 标签栏自定义组件开发、表单状态判断；
- 本地轻量级存储（Preferences）保存用户信息（贴合数据管理教学）。

### 2.4 首页（核心主页面）

**页面结构**：沉浸式导航栏 + 顶部搜索栏 + 功能快捷入口 + 核心模块卡片 + 底部 Tab 导航栏，背景为 #F5F7FA 浅灰

**各区域设计**：

- **沉浸式导航栏**：
  - 背景：透明渐变，跟随页面滚动变化；
  - 左侧：无图标；
  - 中间：标题「校园助手」（30sp，鸿蒙蓝，字重 600）；
  - 右侧：消息图标（铃铛，鸿蒙蓝），有未读消息时显示红色小圆点（暖橙），点击跳转到消息中心页。

- **顶部搜索栏**：
  - 位置：导航栏下方，间距 16px，宽度 95%，居中；
  - 样式：圆角 24px，白色背景，左侧搜索图标（鸿蒙蓝），中间占位符「搜索校园信息 / 课程 / 美食」（24sp，文字辅色），右侧语音图标（鸿蒙蓝，简易版，仅显示无功能）；
  - 交互：点击搜索栏跳转到搜索结果页，支持输入关键词搜索。

- **功能快捷入口**：
  - 布局：3 行 4 列，九宫格 + 1 个更多按钮，每个入口为圆形卡片（64px 直径），间距 16px，整体居中；
  - 内容：图标（鸿蒙蓝，32px）+ 文字（24sp，文字主色，图标下方居中），包含「校园公告」「课程表」「成绩查询」「失物招领」「校园地图」「图书馆」「校园美食」「音乐播放」「待办提醒」「更多功能」；
  - 交互：点击任意入口跳转到对应功能页面，图标有点击放大微动画。

- **核心模块卡片**（垂直排列，每个卡片为白色圆角卡片，宽度 95%，居中，间距 16px）：

  **模块 1：校园公告**
  - 卡片标题：左侧「校园公告」（28sp，鸿蒙蓝，字重 600），右侧「更多」（24sp，鸿蒙蓝）；
  - 公告列表：显示 3 条最新公告，每条包含标题（26sp）+ 发布时间（22sp）+ 简要内容（24sp，一行省略），每条间距 12px。

  **模块 2：今日课程**
  - 卡片标题：「今日课程」（28sp，鸿蒙蓝，字重 600）；
  - 课程内容：无课程时显示「今日无课程，好好休息吧～」（26sp，文字辅色，居中）；有课程时显示课程名称、上课时间、上课地点，暖橙标签标记当前正在上课的课程。

  **模块 3：天气提醒**
  - 布局：左侧天气图标（48px）+ 温度（32sp，文字主色，如「25℃」），右侧天气状况（26sp，文字辅色，如「晴 微风」）+ 校园所在地区（24sp，文字辅色）；
  - 交互：点击天气卡片跳转到天气详情页（简易版，显示未来 3 天天气）。

- **底部 Tab 导航栏**：
  - 布局：5 个 Tab，平分宽度，高度 56px，白色背景，无阴影；
  - 每个 Tab：图标（28px）+ 文字（22sp），未选中时图标和文字为文字辅色，选中时为鸿蒙蓝；
  - Tab 内容：「首页」「校园信息」「学习助手」「生活服务」「我的」；
  - 交互：点击 Tab 切换对应页面，无页面跳转，为组件内切换，有淡入淡出过渡动画。

**交互逻辑**：
- 页面支持下拉刷新，刷新时显示鸿蒙蓝环形进度条；
- 滚动页面时，导航栏从透明渐变变为白色，标题颜色从鸿蒙蓝变为文字主色；
- 核心模块卡片支持上拉加载更多；
- 底部 Tab 导航栏常驻，所有页面均可通过 Tab 快速切换。

**技术要点**：
- 沉浸式窗口管理、Tab 导航栏自定义开发（贴合 Stage 模型、导航教学）；
- 列表组件、卡片组件使用，下拉刷新、上拉加载功能实现；
- 图标组件、微动画实现，页面滚动监听。

### 2.5 校园信息页

**页面结构**：导航栏 + 分类标签栏 + 内容列表 + 底部 Tab 导航栏（常驻）

**各区域设计**：

- **导航栏**：非沉浸式，白色背景，左侧返回箭头（鸿蒙蓝），中间标题「校园信息」（30sp，文字主色，字重 600），右侧无图标。

- **分类标签栏**：
  - 布局：横向滚动，无边界，左侧对齐，每个标签为圆角按钮（高度 36px，内边距左右 16px）；
  - 内容：「校园公告」「校园资讯」「社团活动」「学术讲座」「校园地图」，默认选中「校园公告」；
  - 样式：选中标签（鸿蒙蓝背景，白色文字），未选中标签（浅灰背景，文字主色）；
  - 交互：点击标签切换对应内容列表，横向滑动可查看更多标签。

- **内容列表**：
  - 布局：垂直排列，每个列表项为白色圆角卡片，宽度 100%，间距 12px，内边距 16px；
  - 通用列表项结构：左侧缩略图（80px × 80px，圆角 8px，无图时显示默认校园图标）+ 右侧内容区域（占比 70%）；
  - 内容区域：标题（26sp，文字主色，一行省略）+ 发布时间 / 来源（22sp，文字辅色）+ 简要内容（24sp，文字辅色，两行省略）；

  **校园地图标签专属布局**：
  - 上方：校园平面地图（宽度 100%，高度 200px，圆角 8px），标注核心建筑；
  - 下方：建筑搜索框（同首页搜索栏样式）+ 核心建筑快捷入口（横向滚动，如「教学楼」「图书馆」「食堂」「宿舍」），点击跳转到对应建筑详情页。

**交互逻辑**：
- 点击列表项跳转到对应内容详情页；
- 内容列表支持下拉刷新、上拉加载更多；
- 校园地图支持双指缩放、单指拖动，点击建筑标注弹出建筑名称提示。

**技术要点**：
- 横向滚动标签栏、列表组件开发；
- 图片组件加载、地图缩放拖动功能实现；
- 页面路由跳转（列表项 → 详情页）。

### 2.6 学习助手页

**页面结构**：导航栏 + 功能标签栏 + 功能内容区 + 底部 Tab 导航栏（常驻）

**各区域设计**：

- **导航栏**：同校园信息页，标题「学习助手」；

- **功能标签栏**：
  - 布局：横向排列，4 个标签平分宽度，无滚动；
  - 内容：「课程表」「成绩查询」「图书馆检索」「学习资料」；
  - 样式：选中标签文字为鸿蒙蓝，底部显示 2px 鸿蒙蓝下划线；未选中为文字辅色，无下划线；

- **功能内容区**（标签切换对应内容，无页面跳转）：

  **课程表（默认选中）**：
  - 顶部：周次选择（左侧「上一周」，中间「202X 年第 X 周」，右侧「下一周」，均为 24sp，鸿蒙蓝）；
  - 主体：周视图（周一至周日，纵向排列），每日列显示课程卡片（暖橙 / 鸿蒙蓝 / 清新绿区分），包含课程名称、上课时间、上课地点；
  - 底部：「编辑课程表」按钮（鸿蒙蓝，小按钮），点击跳转到课程表编辑页。

  **成绩查询**：
  - 上方：学号 / 姓名显示（26sp，文字主色），「刷新成绩」按钮（鸿蒙蓝，小按钮）；
  - 下方：成绩列表，每个列表项包含课程名称、成绩、绩点、考试时间，支持按绩点排序。

## 三、页面跳转关系

所有页面遵循"主页面 Tab 切换，子页面返回箭头 + 路由跳转"的逻辑，贴合 HarmonyOS NEXT 页面路由教学要求，核心跳转关系如下：

- 启动页 → 引导页（首次）/ 首页（非首次）；
- 引导页 → 登录 / 注册页 → 首页；
- 首页 → 校园信息 / 学习助手 / 生活服务 / 我的（Tab 切换），首页 → 搜索结果页 / 消息中心页（按钮跳转）；
- 校园信息页 → 各内容详情页 / 校园地图详情页（列表 / 按钮跳转）；
- 学习助手页 → 课程表编辑页 / 图书详情页（按钮 / 列表跳转）；
- 生活服务页 → 失物招领发布页 / 本地音乐列表页（按钮跳转）；
- 我的页 → 个人信息编辑页 / 设置页 / 各功能子页面（按钮 / 图标跳转）；
- 所有子页面均通过左上角返回箭头回到上一级页面，支持侧滑返回（HarmonyOS 原生功能）。

## 开发适配说明

- 严格遵循 HarmonyOS NEXT 设计规范与原子化服务标准；
- 所有页面均采用 HarmonyOS NEXT 的 **ArkUI 声明式开发范式**，使用 ArkTS 语言编写，贴合课程大纲教学内容；
- 布局优先使用线性布局（Row/Column）和相对布局（RelativeContainer），保证移动端自适应，无固定尺寸硬编码；
- 所有图片、音频资源均使用本地资源，无需后台服务器，降低开发难度，贴合初学者开发能力；
- 数据存储均采用 **HarmonyOS NEXT 本地存储方式**（Preferences 轻量级存储 + JSON 文件存储），覆盖课程大纲数据管理教学点；
- 所有组件均基于 **ArkUI 原生组件**开发，少量自定义组件（如课程表、Tab 栏）贴合课程大纲自定义组件教学要求，无过度复杂封装。
