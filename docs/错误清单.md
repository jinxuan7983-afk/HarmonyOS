## P1 — Information（废弃 API，建议修复）

| 废弃 API | 替代方案 | 影响文件数 |
|----------|----------|------------|
| `router.pushUrl(options)` | `router.pushUrl({ url, params }, callback)` 新签名 | 10+ |
| `router.replaceUrl(options)` | `router.replaceUrl({ url, params }, callback)` 新签名 | 5+ |
| `router.back(options)` | `router.back({ url })` | 10+ |
| `router.getParams()` | 使用 `Navigation` 路由参数 | 5+ |
| `getContext(this)` | `this.getUIContext().getHostContext()` | 5+ |
| `animateTo()` | `animateToImmediately()` | 1 |
| `notificationManager.requestEnableNotification()` | 保留但加注释 | 1 |
| `onScroll(xOffset, yOffset)` | `onScroll((scrollOffset, scrollState) => {})` | 1 |
| `textDecoder.decodeWithStream()` | 保留但加注释 | 1 |
